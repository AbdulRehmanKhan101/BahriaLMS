
# Bahria LMS

**Bahria LMS** is a desktop-based Learning Management System built using **C++** and **Qt6 Widgets**. It provides an intuitive interface for managing courses, students, and academic resources. The project uses **QSS (Qt Style Sheets)** for modern UI styling.
